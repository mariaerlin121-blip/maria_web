# maria_web
Website portofolio sederhana.

## Cara menggunakan

1. Buka `index.html` di peramban web Anda.
2. Atau jalankan server lokal sederhana, misalnya:
   - `python3 -m http.server` di dalam folder proyek
   - lalu buka `http://127.0.0.1:8000`

## Isi proyek

- `index.html` - halaman portofolio utama
- `styles.css` - gaya tampilan responsif
- `script.js` - interaksi tombol kontak
